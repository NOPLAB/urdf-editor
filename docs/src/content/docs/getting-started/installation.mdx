## Web Application

The easiest way to use RK is through the web application. No installation required!

[Launch Web App](/rk/app/)

## Desktop Application

Download the latest release for your operating system from the [GitHub Releases](https://github.com/NOPLAB/rk/releases) page.

### System Requirements

- Windows 10/11, macOS 10.15+, or Linux
- GPU with WebGPU/Vulkan support
- 4GB RAM minimum

## Building from Source

### Prerequisites

- Rust 1.85 or later
- For WASM builds: `wasm32-unknown-unknown` target and `trunk`

### Build Commands

```bash
# Clone the repository
git clone https://github.com/NOPLAB/rk.git
cd rk

# Build native desktop app
cargo build --release

# Build WASM web app
cd crates/rk-frontend
trunk build --release
```
