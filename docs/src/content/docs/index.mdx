## What is RK?

RK is a modern 3D CAD editor. It provides the tools you need to create, edit, and visualize 3D models efficiently.

## Features

<CardGrid stagger>
  <Card title="Visual 3D Editor" icon="pencil">
    Intuitively edit links, joints, and visual properties in an interactive 3D viewport. No manual XML editing required.
  </Card>
  <Card title="Real-time Preview" icon="laptop">
    See your changes instantly. Adjust joint positions, modify geometry, and preview kinematics in real-time.
  </Card>
  <Card title="Cross-Platform" icon="rocket">
    Run natively on Windows, macOS, and Linux, or use the WebAssembly version directly in your browser.
  </Card>
  <Card title="Mesh Support" icon="document">
    Import and export STL, OBJ, and DAE mesh files. Visualize complex robot geometries with ease.
  </Card>
</CardGrid>
