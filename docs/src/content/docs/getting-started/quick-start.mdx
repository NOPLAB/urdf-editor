## Creating Your First Robot

1. Launch the application
2. Use **File > New** to create a new assembly
3. Add parts using the Part List panel
4. Connect parts with joints
5. Save your work with **File > Save**

## Interface Overview

The application consists of several panels:

- **Viewport**: 3D view of your robot
- **Part List**: Hierarchical view of all parts
- **Properties**: Edit selected part properties
- **Joint List**: Manage joints between parts
